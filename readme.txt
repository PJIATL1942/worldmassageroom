The Knowband Mobile App Builder converts your online store into a pair of native Android & iOS apps without any coding.

== Description ==
The Knowband Mobile App Builder converts your online store into a pair of native Android & iOS apps without any coding. Entire store inventory & database gets automatically synced on the mobile apps. The WooCommerce Mobile app supports all sorts of languages, currencies, products, shipping methods, payments etc. The apps have a fully dynamic home screen which can be self-designed from extension backend. Inbuilt Zopim & WhatsApp chat options offers reliable communication medium between the admin and app shoppers. Swift login options like Google, Facebook, Phone No (OTP) & Fingerprint are available in the apps.

== Installation ==
1. After purchasing the Knowband Mobile App Builder Module, you will get the following files as listed below:
   a. Compressed file in ZIP format (It contains the source code of the plugin)
   b. User Manual
2. Go to the dashboard of WordPress->Plugins->Add New
3. You will be redirected to a new interface. Click Upload Plugin.
4. Next window will ask you to upload the file. Click on Choose file and Upload the zipped file.
